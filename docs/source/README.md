# docs/source — 一次資料アーカイブ（Hazama 物語の背骨）

ユーザー（note: **quiet_briony5600** 「AIと仕事と趣味のはざま」）自身の著作物を、ゲーム化
（M2 以降）の reference としてアーカイブしたもの。`scripts/` が読む runtime データではなく、
**創作の出典**です。機械可読の索引は [`_manifest.json`](_manifest.json)。

## 取り扱い（重要・デリケートな一次資料）

- 出典は「ハーネスが緩かった頃のAI対話」由来。飛躍・生々しさ＝"タガが外れた"質感と、
  ユーザーの内面吐露が芯。
- adaptation（ゲーム本文化）では、**この生っぽい一人称の声を殺さない**。整形・要約で漂白しない。
  引用・翻案の単位は [../M2-spine.md](../M2-spine.md) で管理する。
- 各ファイルは HTML→Markdown 自動整形。**本文化に使う前に、必ず原文（url）か本アーカイブを
  直接読み、自動整形の崩れ・脱落を確認する**こと（記号・改行・引用の欠けが起こりうる）。

## 収集元と方法

- プロフィール: https://note.com/quiet_briony5600

  **投稿日昇順＝概念の積み上げ順**（ファイル名先頭の 01〜30）。
