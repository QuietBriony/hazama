① Safer-AGI: A Framework for Inherently Safer AGI through Language-Mediated Active Inference

リンク：
https://arxiv.org/abs/2508.05766

要旨（実データ ingest に基づく / 簡潔版）
｜ LLM を “アクティブ推論（Active Inference, AIF）” と統合し、
｜ AGIが自己の信念・価値を 言語で表現 しながら行動し、

という新しい安全フレームワーク。

重要ポイント
	•	後付けの安全策（RLHF → ガードレール方式）は限界
	•	“価値の表現” を内部状態として扱うべき
	•	推論の安定＝free-energy 最小化
	•	認知均衡の崩壊が危険性の起点

	•	Void（安定場）＝ free-energy 最小

	•	創観（創造構造）＝安全性を“外付け”ではなく“構造”として持つ

https://arxiv.org/abs/2501.15280
